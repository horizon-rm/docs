## 1. 云台控制


云台控制的目标是：**让枪口稳定指向目标**。这里分为两种模式。

### 1.1 模式一：普通双环角度反馈
**应用场景**：仅使用编码器作为位置反馈（无IMU或IMU失效时），或者用于俯仰轴（Pitch）控制。

**原理**：

- **内环（速度环）**：控制电机转速，输出电流（转矩）。

- **外环（位置环）**：控制电机到达指定角度，输出期望速度给内环。

**结构**：
`目标角度` -> [位置PID] -> `目标速度` -> [速度PID] -> `电流/电压` -> 电机 -> `实际角度/速度`


### 1.2 模式二：基于IMU坐标系的云台控制

**应用场景**：当底盘旋转时，云台需要通过 IMU（陀螺仪）的角速度数据前馈或闭环，实现绝对的“稳像”或“指哪打哪”。在 RM 中，云台 Yaw 轴电机通常连接在底盘上。如果底盘转动，云台电机必须反向转动来补偿，才能保持枪口在世界坐标系中静止。

### 1.2.1 双环 PID 结构
与普通编码器双环控制不同，这里采用 **IMU 角度作为位置环反馈**，**IMU 陀螺仪角速度作为速度环反馈**。这样做的好处是：  
- 位置环直接闭环枪口的绝对姿态，不受底盘扰动影响。  
- 速度环使用陀螺仪数据，响应快、抗干扰能力强，且避免了电机编码器速度反馈的延迟。

**控制框图**  
```
目标角度 ──→ [位置PID] ──→ 目标角速度 ──→ [速度PID] ──→ 电流指令 ──→ 云台电机
    ↑                          ↑
    |                          |
IMU 角度反馈            IMU 角速度反馈
```


### 1.2.2 代码实现解析

以 Pitch 轴为例（Yaw 轴同理）：

```c
// 位置环（角度环）
PID_Calculate(&MOTOR->DM4310_Pitch.PID_P, 
              IMU_Data.pitch,                       // 反馈：IMU俯仰角
              MOTOR->DM4310_Pitch.DATA.Aim);        // 目标：云台期望俯仰角

// 速度环
PID_Calculate(&MOTOR->DM4310_Pitch.PID_S,
              IMU_Data.gyro[1] * 50.0f,             // 反馈：IMU俯仰角速度（乘以系数转换单位）
              MOTOR->DM4310_Pitch.PID_P.Output);    // 目标：位置环输出
```

**关键点解释**  

- **位置环反馈**：`IMU_Data.pitch` 直接读取 IMU 解算后的俯仰角（世界系），无需再通过电机编码器换算。  

- **速度环反馈**：`IMU_Data.gyro[1]` 是 IMU 测量的俯仰轴角速度（通常单位为 rad/s），乘以 `50.0f` 可能是为了将单位转换为电机控制所需（例如转换为 deg/s 或适配 PID 参数）。实际工程中应根据 IMU 数据手册和电机控制量纲调整系数。  

- **位置环输出**直接作为速度环的目标值，实现串级控制。

Yaw 轴代码结构相同，但使用 `IMU_Data.YawTotalAngle` 作为位置环反馈（累计角度，避免跳变），内环反馈使用 `QEKF_INS.Gyro[2] * 50.0f`（偏航角速度）。

```c
// Yaw 轴位置环（角度环）
PID_Calculate(&MOTOR->DM4310_Yaw.PID_P,
              IMU_Data.YawTotalAngle,                // 反馈：IMU累计偏航角
              MOTOR->DM4310_Yaw.DATA.Aim);           // 目标：云台期望偏航角

// Yaw 轴速度环
PID_Calculate(&MOTOR->DM4310_Yaw.PID_S,
              QEKF_INS.Gyro[2] * 50.0f,              // 反馈：IMU偏航角速度
              MOTOR->DM4310_Yaw.PID_P.Output);       // 目标：位置环输出
```

### 1.2.3 关于角度反馈的注意事项

- **Yaw 轴累计角度**：`YawTotalAngle` 是一个累计量（通常由 IMU 的角速度积分得到），不存在 360° 跳变问题，因此可直接用于位置环。如果使用归一化角度（如 ±180°），则必须额外处理跳变。  
- **Pitch 轴范围限制**：俯仰角通常有机械限位（如 -15° ~ +30°），位置环输出需限幅，防止超出范围。  
- **坐标系对齐**：确保 IMU 安装方向与电机控制方向一致（例如 IMU 的 X 轴指向枪口，Z 轴向上），否则需在软件中进行轴映射。

### 1.2.4 前馈与扰动抑制

在双环 PID 基础上，可叠加角速度前馈以进一步提升抗扰动能力。例如当底盘突然转动时，IMU 检测到角速度突变，可直接将该项前馈叠加到速度环输出，使电机更快响应补偿。

```c
// 可选的前馈项
float feedforward = K_ff * (QEKF_INS.Gyro[2] * 50.0f);
float final_output = PID_S.Output + feedforward;
```

### 1.2.5 调试建议

1. **先调内环（速度环）**：给定期望角速度（如 200°/s），观察陀螺仪反馈是否平稳跟随。内环应做到快速响应、无超调。  
2. **再调外环（位置环）**：给定一个角度阶跃信号，观察枪口响应。位置环应保证稳态误差小、动态过程平滑。  
3. **引入前馈**：在底盘旋转时，开启前馈补偿，检查枪口是否仍然稳定指向目标。  
4. **注意单位转换**：确保 `gyro * 50.0f` 后的值域与 PID 参数匹配。如果出现异常震荡，首先检查量纲是否正确。
